# PairNumbersToSum

The app finds pairs of integers from a list that sum to a given value. 
The app will take as input the list of numbers as well as the target sum.


Example Input
1,9,5,0,20,-4,12,16,7 
12

Example Output
[ [ 0, 12 ], [ -4, 16 ], [ 5, 7 ] ]


## Installation
1. Clone the project.
2. npm install

## Run 
1. Run the index.js file with node
2. Type your inputs
3. Get your results
